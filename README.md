## 设计理念

优美优于丑陋，明了优于隐晦
简单优于复杂，复杂优于繁杂
扁平优于嵌套，稀疏优于稠密
可读性很重要！
特例亦不可违背原则，即使实用比纯粹更优。
错误绝不能悄悄忽略，除非它明确需要如此。
面对不确定性，拒绝妄加猜测。
任何问题应有一种，且最好只有一种，显而易见的解决方法。

以上节选自《Python之禅》
